# useScrollViewOffset

This hook allows you to create animations based on the offset of a `ScrollView`.
The hook automatically detects if the `ScrollView` is horizontal or vertical.

## Reference

```tsx
import { useAnimatedRef, useScrollViewOffset } from 'react-native-reanimated';

function App() {
  const animatedRef = useAnimatedRef<RefObject<ScrollView>>();
  const scrollOffset = useScrollViewOffset(animatedRef);

  return <ScrollView ref={animatedRef}>{/* ... */}</ScrollView>;
}
```

<details>
<summary>Type definitions</summary>

```typescript
function useScrollViewOffset(
  animatedRef: AnimatedRef<AnimatedScrollView>,
  initialRef?: SharedValue<number>
): SharedValue<number>;
```

</details>

### Arguments

#### `animatedRef`

An Animated reference to a `ScrollView`. The reference should be passed to the
appropriate `ScrollView` in the `ref` prop.

#### `initialRef` <Optional/>

An optional shared value to be updated with the scroll offset. If not provided
a new shared value will be created internally.

### Returns

A shared value which holds the current offset of the `ScrollView`.
